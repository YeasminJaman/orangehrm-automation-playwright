# OrangeHRM Automation – Playwright + TypeScript

This project contains 4 critical automated UI tests for OrangeHRM:
1. Successful Login
2. Invalid Login
3. Add New Employee
4. Logout

## Tech Stack
- Playwright
- TypeScript
- Page Object Model
- HTML Reporter

## Project Structure
/pages      → Page Objects  
/tests      → Test Specs  
/utils      → Test Data  

## ▶️ Run Tests
npm install  
npx playwright test  

## View Report
npx playwright show-report
